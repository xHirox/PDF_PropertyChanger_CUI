・インストール：不要
itextsharp.dllと同じ階層でPDF_PropChgSample_CUI.exeを実行する
・アンインストール：不要
該当のフォルダごと削除する
レジストリ書き換え等は一切なし

・使用方法・動作
1.　パスワードを設定したいPDFファイルが格納されているパスを入力する（パスは絶対パス）
　　#該当のフォルダをコンソールにドラッグアンドドロップすることで絶対パスが入力できます
2.　指定パス内に格納されているPDFファイルを取得し表示する
　　#指定パス内にPDFファイルが存在しない場合は処理を中断します
3.　PDFファイルに設定したいパスワード文字列を入力
4.　概要タブの項目「タイトル」「作成者」「サブタイトル」「キーワード」を入力
5.　パスワード・プロパティを各PDFファイルに設定し、動作終了
　　（Prop set done.が表示されれば該当ファイルは正常終了）
5.　すべての処理が終了すると「終了しました」を表示する

※　ユーザー・オーナーパスワード（閲覧・編集パスワード）が設定されているPDFファイルは、
　　例外をcatchして処理しません
※　パスワード以外でもPDFファイルが正常に開けない、その他何かしらの例外をcatchした場合は、
　　パスワード・プロパティ設定処理はせずに次のPDFファイルへ処理が移ります
※　例外が発生した場合は「Exception/エラーメッセージ」が表示されます

・注意点
1.　ユーザー（閲覧）パスワード、オーナー（編集）パスワードが設定されていないPDFファイルが対象です
2.　PDF文書のプロパティ「概要」タブの情報は、変更不要の場合は変更前の値をそのまま引き継ぎます
